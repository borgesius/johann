# Electron Spec Brief

Build a design-led desktop workspace for shaping product ideas into shippable execution plans.

The repo should feel like something a strong engineer could pick up and continue:
- clear entry points
- a tasteful but not generic shell
- lightweight smoke coverage
- docs that explain how to work on it and how packaging would be approached

Prefer a structure that can evolve into a real product rather than a one-file demo.
